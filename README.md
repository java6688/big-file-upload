# 大文件上传 Demo

## 项目简介
本项目演示了如何在前端实现大文件分片上传，支持断点续传、并发上传、秒传等功能，后端基于 Node.js + Express + Multer 实现。

## 技术栈
- 前端：原生 HTML + JavaScript（ES6）
- 后端：Node.js + Express + Multer
- 数据库：SQLite（记录文件上传状态）
